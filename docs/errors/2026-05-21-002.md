# 002: QWidget C++ 对象已删除（PDF 模式切换）

**日期：** 2026-05-21
**模块：** `src/mbforge/ui/pdf_viewer.py`
**状态：** 已解决

## 错误描述

```
RuntimeError: wrapped C/C++ object of type QWidget has been deleted
```

在 PDF 查看器中切换单页/连续模式时触发。

## 根因分析

`QScrollArea.setWidget()` 会接管旧 widget 的所有权并可能删除它。当从连续模式切换到单页模式时，`continuous_widget` 被 Qt 删除。再切回连续模式时，Python 仍引用已删除的 C++ 对象。

## 解决方案

在 `_render_continuous()` 中每次重建 `continuous_widget` 和 `continuous_layout`，确保始终使用新鲜的 QWidget。

```python
def _render_continuous(self):
    self._clear_pages()
    # 每次重建，避免引用已删除的 C++ 对象
    self.continuous_widget = QWidget()
    self.continuous_layout = QVBoxLayout(self.continuous_widget)
    ...
    self.scroll.setWidget(self.continuous_widget)
```
