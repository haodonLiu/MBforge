### 🎯 核心发现（Top 3 Critical）

#### 1. **测试覆盖率极低** (P0)
- **现状**: 133 个 Python 文件仅 28 个测试（~21%）
- **影响**: 19 个 API 路由零测试，任何重构都可能引入隐蔽 bug
- **修复**: 1 周内补齐路由冒烟测试（57 tests），覆盖率升至 30%

#### 2. **模型首次加载 30s 无预热** (P0)
- **现状**: `server.py:_prewarm()` 是空函数，用户首次请求卡死
- **影响**: MolDet + MolScribe 首次加载 30s，无进度提示
- **修复**: 实现并发预热 + 前端轮询健康检查（2 小时）

#### 3. **代码质量债务累积** (P1)
- **现状**: 254 个 Ruff lint 问题（未使用导入、命名不规范）
- **影响**: 可维护性下降，新人 onboarding 困难
- **修复**: 自动修复 142 项 + 人工审查 112 项（4 小时）
