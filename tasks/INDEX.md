## 并行执行策略

### Wave 1（立即启动，3 路并行）

- **Task A** — 数据库抽象层（基础，阻塞 B）
- **Task C** — 可观测性层（独立）
- **Task D** — 管线优化（独立）
- **Phase 1 零碎修复** — chematic 验证 + Mutex 迁移 + TODO 清理

### Wave 2（A 完成后）

- **Task B** — 分子三层迁移（依赖 A 的 Table trait）

### 总预计时间

- 串行: 3-4 + 2-3 + 2-3 + 3-5 = 10-15 天
- 并行: max(A+B, C, D) = max(5-7, 2-3, 3-5) = 5-7 天
